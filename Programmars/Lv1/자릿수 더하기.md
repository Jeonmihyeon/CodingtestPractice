### 파이썬은 몫만 구하기 위해서 나눗셈 연산자 "//"을 사용한다. "/"은 소수점 자리까지 나온다!


```
def solution(n):
    answer = 0

    while n >= 10:
        answer += n % 10
        n //= 10;
    answer += n

    return answer
```
